## 概述

`@kit/cmd`是一个命令菜单React组件库，也可以用作搜索框显示。此库不支持自动过滤和排序，需要手动处理。此库只有组件行为，需要样式请自行添加。

此库灵感和部分代码来源：[cmdk](https://github.com/pacocoursey/cmdk)

## 安装

暂未发布

```bash
pnpm add @kit/cmd
```

import { Command } from '@kit/cmd'
